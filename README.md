
#  Real Estate Sales Performance Dashboard ‚Äì Excel Project

##  Project Overview

This project is an **interactive Real-Estate Sales Dashboard** built using **Microsoft Excel**.
It provides a comprehensive view of property sales performance across different cities, agents, and property types.

The dashboard helps management to track:

* Sales growth
* Agent efficiency
* Property-type revenue
* City-wise revenue trends
* Monthly sales performance

---

##  Dashboard KPIs

| Metric                      | Description                     |
| --------------------------- | ------------------------------- |
| **Total Property Sold**     | Total number of sold properties |
| **Total Properties**        | Overall properties available    |
| **Total Unsold Properties** | Remaining unsold units          |
| **Conversion Rate**         | Sold / Total Properties         |

##  Interactive Filters (Slicers)

| Filter Type       | Options                                          |
| ----------------- | ------------------------------------------------ |
| **Building Type** | Condo, Detached, Office Building, Ranch          |
| **Agent Name**    | All sales agents                                 |
| **Location**      | Austin, Dallas, Fort Worth, Houston, San Antonio |

These slicers dynamically update all charts and KPIs.

##  Key Business Insights

* Houston generates the **highest revenue** among all cities.
* Houses outperform Apartments and Commercial properties in total revenue.
* Peak sales observed around **July ‚Äì August**.
* Top performing agents consistently exceed monthly targets.
